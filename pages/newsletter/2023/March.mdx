## <a id="news-explore">News & Explore</a>

### JS import maps are available on major browsers

ES modules modernize JavaScript code inclusion and reuse in web applications, offering advantages over older methods. Utilize the tag to easily map external module names to URLs for efficient integration in your code. It's now supported cross-browser!
[Web.dev](https://web.dev/import-maps-in-all-modern-browsers/)

### Qwik

An HTML-first framework that reduces JS usage is about to release its 1.0 and their candidate 1.0 version is out.
[1.0 RC Doc](https://www.builder.io/blog/qwik-rc-milestone)

### Static Exports on Next.js

Next.js enables starting as a static site or Single-Page Application (SPA), then later optionally upgrading to use features that require a server.
[Static Export](https://beta.nextjs.org/docs/configuring/static-export)

### Visual Guide for Node.js Event Loop

The Node.js event loop may be challenging to grasp, even for experienced developers. This visual guide aims to clarify the event loop and enhance your understanding of its inner workings.
[Nodejs Event Loop](https://www.builder.io/blog/visual-guide-to-nodejs-event-loop)

### GitHub Copilot X

AN AI pair programmer using OpenAI's Codex model. Now introducing GitHub Copilot X, this AI assistant is accessible throughout the development lifecycle, improving productivity by reducing manual tasks and enabling developers to focus on innovation. It's not just Code Completion tool but helper along the journey.
[Copilox X Announcement](https://github.blog/2023-03-22-github-copilot-x-the-ai-powered-developer-experience)

### React Docs gets an update!

React docs have been worked on for years and it's now live with the new version.
[React Docs](https://react.dev)

### Typescript 5.0 is out!

Bringing new features like: "decorators", "const type", "module resolution bundler" and many more.
[Typescript 5.0](https://devblogs.microsoft.com/typescript/announcing-typescript-5-0/)

### Story of Success, Discord storing Trillions of Messages

You read it correctly, "Trillions"... Back then Discord was storing ["Billions" of messages](https://discord.com/blog/how-discord-stores-billions-of-messages), and now they are storing Trillions of Messages. How? Well, we have to check it out.
[How Discord Stores Trillions of Messages](https://discord.com/blog/how-discord-stores-trillions-of-messages)

### Scoring Frameworks on 2023

There's many frameworks out there and they all solve problems we are facing everyday. But at what cost?
[Web Framework Performance Report](https://astro.build/blog/2023-web-framework-performance-report/)

### Running Hugging Face from JS

Did you know you can run Hugging Face (Open source AI community) models from JS?.
[HuggingFaceJs](https://github.com/huggingface/huggingface.js)
